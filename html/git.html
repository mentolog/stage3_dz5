Git

  • git -v - посмотреть номер установленной версии Git под Windows;
  • git init - инициализация репозитория в папке под Windows;
  • git clone - скопировать удаленный репозиторий на локальный диск;
  • git status - состояние файлов которые необходимо выгрузить в удаленный репозиторий;
  • git add - добавить файл для отслеживания его локальным репозиторием;
  • git commit -m "first_commit" - сохранить измененные или новые файлы в проекте на удаленный репозиторий;
  • git push - окончательно подтвердить сохранение изменений в удаленный репозиторий;
  • git config --list - посмотреть конфигурацию локального Git;
  • git config credential.username "sky-best-student" - добавить в конфиг параметр для авторизации;
  • git log - посмотреть ранее выполнявшиеся коммиты;
  • git pull - получить измененные файлы с удаленного репозитория в локальный;
  • git branch first_branch - создать ветку проекта;
  • git checkout first_branch - переключиться в ветку проекта;
  • git restore --staged - отменить наблюдение и отслеживание изменений в файле локального репозитория;
